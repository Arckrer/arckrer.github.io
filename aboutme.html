ကျနော်က Bun Gon ဖြစ်ပြီး coding ရေးတာနဲ့ အိုင်တီနည်းပညာအပါအဝင် ဘာသာရပ်အချို့ကို စိတ်ဝင်စားပါတယ်။
တက္ကသိုလ်ဝင်တန်း အောင်မြင်ခဲ့တုန်းက ကွန်ပျူတာတက္ကသိုလ် ဝင်ခွင့်အတွက် ရမှတ် ၁၀ မှတ်နီးပါး လိုနေခဲ့လို့ တက်ခွင့်မကြုံခဲ့သလို ကျေးလက်တောရွာမှာ နေထိုင်တာကြောင့် ကွန်ပျူတာသင်တန်း ဘာတစ်ခုမှ မတက်ခဲ့ဖူးပါဘူး။ ဒါပေမယ့် လက်လှမ်းမီသလောက် ကွန်ပျူတာဆိုင်ရာ စာအုပ်စာတမ်းအပြင် တခြားစာပေတွေကို လေ့လာဖတ်ရှုခွင့် ပုံမှန်ရခဲ့တာကြောင့် အိုင်တီဆိုင်ရာ အသိသညာအချို့ကို ရင်းနီးနေခဲ့သလို ရှိနေတာပါ။ ၂၀၁၁ မှာ ကွန်ပျူတာစသုံးဖူး၊ ၂၀၁၈ မှာတော့ mobile internet ကို စတင်ထိတွေ့ခွင့်ရချိန်ကစပြီး အိုင်တီပညာရပ်ကို ပိုမိုထိတွေ့လေ့လာခွင့် ရလာပါတယ်။ ပထမ အစောပိုင်းတုန်းက သူလိုငါလိုပဲ Facebook လူမှုကွန်ယက်ပေါ်မှာ စာတိုပေစ ရေးလာခဲ့ပေမယ့် ကြာလာတော့ စိတ်တိုင်းကျ အားရမှုမရှိသေးတာကြောင့် Facebook page ဖွင့်ပြီးရေးလာတဲ့အခါမှာ စာပေကုန်ကြမ်း ပိုမိုလိုအပ်လာတဲ့အတွက် Wikipedia တို့ Quora digest တို့လို နာမည်ရ ဝက်ဘ်ဆိုက်အပါအဝင် blogger သမားတွေရဲ့ ဝက်ဘ်ဆိုက်စာမျက်နှာအချို့ကို တခုတ်တရ လေ့လာမှတ်သားဖြစ်လာတယ်။ မြန်မာပြည်က လူငယ်တွေရေးတဲ့ ဘလော့ဂ်တွေကို လိုက်ဖတ်ပြီး စိတ်ဝင်စားတဲ့နေရာတွေမှာ comment ဝင်ပေးရင်းက mminsight မှာ စာတစ်ပုဒ်စ နှစ်ပုဒ်စ ဝင်ရေးခွင့်ကြုံပြီး WordPress အကြောင်း နည်းနည်းသိစ ပြုလာပါတယ်။ insight for Myanmar Facebook စာမျက်နှာမှာ ရံဖန်ရံခါ ဝင်ရေးကူပေးရင်း python လို programming language ကို Sololearn မှာ ဝင်လေ့လာဖြစ်သွားတာကစပြီး HTML, CSS, JavaScript အပါအဝင် programming language တချို့ကို ဆက်တိုက်လေ့လာဖြစ်ခဲ့ပါတယ်။ ဒီမတိုင်ခင်မှာတော့ MOOC မှာ online certificate ပေးတဲ့ နိုင်ငံတကာတက္ကသိုလ်တွေရဲ့ course တချို့ကို သင်ယူဖြစ်ပါသေးတယ်၊ အဲဒါတွေက ငွေသွင်းမှ certificate ထုတ်ပေးတာကြောင့် အတွေ့အကြုံအနေနဲ့ပဲ ရှိခဲ့တာပါ။ Sololearn မှာက course တစ်ခုပြီးတိုင်း certificate တစ်ခုကို email နဲ့ပို့လာပေးပါတယ်၊ အခပေးစနစ်နဲ့သင်ရင်တော့ ပိုစုံမလားမသိပါ။
တကယ်တမ်း ဝက်ဘ်ဆိုက်ရေးကြည့်တဲ့အခါမှာ w3school ကိုပိုပြီး အလေးထားလေ့လာရတာပါ၊ သူက အခကြေးငွေနဲ့ certificate ရောင််းချပေးတာ၊ ငွေပေးပြီး လေ့လာနိုင်ရင် ပိုကောင်းတဲ့ ပညာရနိုင်တာရှိပါတယ်။ တခြား အားထားရတဲ့ ဝက်ဘ်ဆိုက်အများကြီး ရှိနေပါသေးတယ်။
